
# 🐧  Day 2 - Variables in Shell Scripting

## What are Variables?

Variables are used to store data in a script.

---

## Defining Variables

```bash
name="Harshit"
age=22
````

---

## Accessing Variables

```bash
echo $name
echo $age
```

---

## Rules

* No spaces around `=`
* Use `$` to access value

---

## Readonly Variable

```bash
readonly name="Harshit"
```
