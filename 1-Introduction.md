# 🐧 Day 1 - Shell Scripting 



## What is Shell Scripting?

Shell scripting is a way to automate tasks in Linux using a sequence of commands written in a script file.

---

## Why Use Shell Scripting?

- Automates repetitive tasks  
- Saves time and effort  
- Helps in system administration  
- Essential for DevOps and automation  

---

## What is a Shell?

A shell is a command-line interpreter that interacts with the Linux kernel.

### Examples of Shells:
- Bash (most common)
- Zsh
- Fish

---

## First Shell Script

Create a file:

```bash
nano script.sh
````

Add the following:

```bash
#!/bin/bash
echo "Hello, DevOps!"
```

Make it executable:

```bash
chmod +x script.sh
```

Run the script:

```bash
./script.sh
```

---

## Explanation

* `#!/bin/bash` → Shebang (tells system to use bash shell)
* `echo` → Prints output to terminal
