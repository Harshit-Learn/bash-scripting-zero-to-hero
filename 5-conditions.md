
# 🐚 Day 5 - If-Else Conditions

## If Statement

```bash
if [ $a -gt 5 ]
then
  echo "Greater"
fi
````

---

## If-Else

```bash id="c5k2pl"
if [ $a -gt 5 ]
then
  echo "Greater"
else
  echo "Smaller"
fi
```
